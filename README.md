# React JS JSX and CSS Mastering – Front-End Development

## Overview
This project is a comprehensive React front-end application built to demonstrate proficiency in **React.js**, **JSX**, and **CSS/Tailwind**. It covers component architecture, state management, hooks usage, API integration, and responsive design principles.

---

## Features
- Fully built with **React.js** using functional components
- **Tailwind CSS** for responsive and modern UI design
- Dynamic pages including:
  - Home
  - About
  - Tasks
  - API Data fetching
- Context API for theme management
- Custom hooks (`useLocalStorage`, `useTheme`) for reusable logic
- Clean folder structure:
```

src/
├─ components/
├─ pages/
├─ hooks/
├─ context/

````
- Vite as the build tool for faster development and hot module reloading

---

## Installation

1. **Clone the repository**

```bash
git clone https://github.com/PLP-MERN-Stack-Development/react-js-jsx-and-css-mastering-front-end-development-adecares-Dev.git
cd react-task-app
````

2. **Install dependencies**

```bash
npm install
```

3. **Run the development server**

```bash
npm run dev
```

4. Open your browser at `http://localhost:5173` (Vite default)
